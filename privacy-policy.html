Privacy Policy — DRS Pin Tool

Effective date: 23.07.2026

DRS Pin Tool is a personal-use application built by the sole owner of Daily Ritual Studio, used only to publish this business's own Pinterest content. It is not distributed to or used by any other person or business.

Data accessed: DRS Pin Tool accesses the authenticated user's own Pinterest account data only — specifically, board information and the ability to create Pins on that account, via Pinterest's official API. It does not access any other Pinterest user's data.

Data storage: Access credentials (OAuth tokens) are stored locally on the developer's own device and are never transmitted to or stored on any third-party server. No Pinterest user data is stored, sold, shared, or used for any purpose beyond creating Pins on this single account.

Data sharing: No data accessed through this app is shared with any third party.

Retention: Access tokens are retained locally until manually revoked by the account owner via Pinterest's account settings, or until the app is discontinued.

Contact: For questions about this policy, contact Daily Ritual Studio via Etsy's messaging system at dailyritualstudioco.etsy.com.
